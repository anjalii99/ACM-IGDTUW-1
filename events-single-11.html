Git, GitHub & Linux

Time: 1-3 PM, 29 March, 2017

Location: SD Lab, IT Block, IGDTUW

ACM IGDTUW brings to you an OpenSource session including Git, GitHub and Linux. 

IMPORTANT NOTES

  • Students should carry their own laptops and a dongle (latter is not necessary, but preferable).
  • Windows users are required to dual boot their systems with LINUX because we’ll be working with the bash shell. Those already using LINUX or MacOS needn’t worry about this point.
  • Everyone is required to create an account on GitHub and install/configure Git in their systems. We’ll be assisting users facing problems in configuring Git during the session.
